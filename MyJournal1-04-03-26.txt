Logical Operators - ✅
    Logical Operator:  |  Description:                |  Example:
    and                |  if x and y are true = True  |  x and y
    or                 |  if x or y are true = True   |  x or y
    not                |  if x is false = True        |  not x

        and - in this logical operator if the two statement are "True" the output is "True"
        so if the one statement becomes "False" even the one is "True" the output will becomes "False".

        or - in this logical operator if the statement are "True" and "False" the output will become "True".

        not - in this logical operator if the statement are "True" if you use this operator, the output will become "False". (vise versa)

Control Flow - 

Loops:
    for
    while
    break
    continue

example Loops:

for:
    for i in range(5):
    print("Hello", i)

example loop:
    for i in range(5):
    print("Hello")

    Output:
    Hello
    Hello
    Hello
    Hello
    Hello

Explanation: range(5) = 0,1,2,3,4 
- So loop runs 5 times.

Example with numbers:
    for i in range(1,6):
    print(i)

    Output:
    1
    2
    3
    4
    5

The stop value (6) is exclusive, meaning it is not included in the range.


while example:
    count = 1

    while count <= 5:
        print(count)
        count += 1
